# Bio Scale Agency

Compliance-first marketing site for peptide and research-compound brands. Built as a static landing page with research-use-first positioning for Google and Meta advertising eligibility.

## Live site

**[https://bioscaleagency.com/](https://bioscaleagency.com/)**

## Repository

[https://github.com/bilalanwar1/peptides](https://github.com/bilalanwar1/peptides)

## Local development

Serve the project folder with any static file server:

```bash
python -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Project structure

```
.
├── index.html      # Full landing page
├── images/         # Logo and product imagery
├── .htaccess       # Apache rewrite / cache rules
└── README.md
```

## Tech

- Static HTML, CSS, and vanilla JavaScript
- No build step required
- Apache `.htaccess` for production hosting
